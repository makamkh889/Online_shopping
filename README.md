# Online Shopping Project
This project is an online shopping platform built using ASP.NET Core MVC 7. It is designed to be a user-friendly and efficient e-commerce website that allows users to browse and purchase products from a variety of categories.

Features
User-friendly interface: The platform is designed with a clean and intuitive interface that allows users to easily browse 
Efficient checkout process: The checkout process is streamlined to minimize the number of steps required to complete a purchase, making it easy for users to buy products quickly and efficiently.
Robust product management: The platform allows sellers to manage their products easily, including adding, updating, and deleting products, as well as setting prices and inventory levels.
Mobile-friendly design: The platform is designed to be mobile-responsive, allowing users to shop on-the-go using their mobile devices.
Future Updates
This project is still in development, and we plan to add several new features in the near future
